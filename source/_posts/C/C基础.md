# 一、C语言基础
## 1.第一个C程序

```c
#include <stdio.h>
int main(){
    printf("Hello World!/n");
    return 0;
}
//std - 标准
//i - 输入
//o - 输出
//int 为整型
//main() - 主函数 其中（）里面写参数
//printf() - 为库函数
//return 0; - 主函数返回值为0
//注意void main(){} 为古老写法，不可用
```
